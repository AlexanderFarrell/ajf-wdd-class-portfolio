Functional Programming

Pure Function Rules:

  1. Absolutely no reliance on values outside the function. So closures which capture are not pure?
  2. Absolutely no side-effects. It doesn't rely or change anything other than the parameters or what is created inside.
  3. Same Parameters -> Same Result
