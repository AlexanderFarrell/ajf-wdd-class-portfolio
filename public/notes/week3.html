Notes:

  • A method can be created by simply giving an object a property which is a function.
  • Object literal is just created, like const a = {}
  • Properties are accessible like an array or with the . operator
  • Undefined is returned if it doesn't have it, not null
  • Computation of a property is possible.
  • Properties can be made with the symbol type
  • in is used to determine if it has the property
  • delete gets rid of properties
  • A namespace can also be used to better maintain scope in JavaScript
  • Math is the method of doing math.
  • There is a Math.hypot() method for the hypotenuse of a right triangle.
  • pattern for regular expressions is with //
  • Character groups done with [aeiou]. [^A-Z] negates, and is a range
  • . any character \w matches any word char \W matches any non-word char \d matches any digit \D non-digit \s whitespace \S non whitespace
  • Document.body - body element
  • Document.images - list of img
  • Document.links - list of a with href
  • Document.anchors - list of a with name
  • Document.forms - list of forms
  • document.getElementsByTagName('li') = $('li')
  • document.getElementsByClassName('hero') = $('.hero')
  • document.querySelector() lets us do stuff like css finds them, returns the first, querySelectorAll() does many, returning empty if none
  • There are text nodes document.createTextNode('Something')
  • .insertBefore()
  • innerHtml doesnt execute scripts
